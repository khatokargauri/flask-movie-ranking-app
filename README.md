Flask Movie Ranking App

A Python Flask web application with an API that allows users to add, rank, and manage their favorite movies. 

Features

1. Add movies with title and poster

2. Rank movies from 1–10

3. View movie details with ratings and synopsis

4. Update or delete entries

5. Responsive UI for easy interaction

Tech Stack

Backend: Python (Flask)

Frontend: HTML/CSS templates

API Integration: REST endpoints
